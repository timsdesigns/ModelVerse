ModelVerse API

The ModelVerse API is a RESTful service that provides developers with access to a database of 3D Models. It includes details on models, properties, and categories, as well as user profiles and favorites.

Developers can integrate the API into their applications, utilizing a MongoDB database and a RESTful architecture. The API is designed for efficiency and ease of integration, allowing for quick development and enhancement of applications.

Features

  • Return a list of ALL models to the user
  • Return data (description, category, property, model URL, whether it's featured or not) about a single model by title to the user
  • Return data about a category (description) by name/title (e.g., "Product")
  • Return data about a property (vertex count, materials, size) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a model to their list of favorites
  • Allow users to remove a model from their list of favorites
  • Allow existing users to deregister

API Reference
